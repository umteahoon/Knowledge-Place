## 음악 파일 요구사항:
- 형식: MP3 (권장)
- 품질: 128kbps 이상
- 길이: 2-5분 권장
- 볼륨: 정규화된 오디오 레벨

## 추가 방법:
1. 음악 파일을 이 폴더에 복사
2. 파일명이 위의 목록과 일치하는지 확인
3. 웹사이트에서 자동으로 로드됨

- 음악은 자동으로 연속 재생됩니다
- 사용자가 볼륨 조절 및 음소거 가능
