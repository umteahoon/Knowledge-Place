# 지식 플레이스 - 교육 게임 플랫폼

재미있는 게임으로 학습하는 스마트 교육 플랫폼입니다.

## 🎮 포함된 게임들

### 📚 교과목별 퀴즈 게임
- **수학** 🔢: 방정식, 기하학, 연산 문제
- **영어** 🇺🇸: 어휘, 문법, 동사 변화  
- **국어** 📚: 문법, 높임법, 고전문학
- **한국사** 🏛️: 고대사부터 근현대사까지
- **세계사** 🌍: 주요 역사적 사건들
- **과학** 🔬: 물리, 화학, 생물, 지구과학

### 🧀 치즈 헌터
뱀 게임과 한국어 퀴즈를 결합한 재미있는 게임입니다.
- 화살표 키 또는 WASD로 뱀 조작
- 치즈를 먹으면 초성 퀴즈 출제
- 다양한 아이템과 효과
- 명예의 전당 시스템

## 🎯 게임 모드

1. **⚡ 스피드 퀴즈**: 60초 시간 제한 도전
2. **💪 서바이벌 모드**: 3개 생명으로 무한 도전
3. **📖 연습 모드**: 시간 제한 없는 학습
4. **🎯 도전 모드**: 어려운 문제로 실력 테스트

## 🚀 배포 방법

### Netlify 배포
1. 이 저장소를 GitHub에 업로드
2. Netlify에서 새 사이트 생성
3. GitHub 저장소 연결
4. 빌드 설정:
   - Build command: `npm run build`
   - Publish directory: `dist`
5. 배포 완료!

### 로컬 실행
```bash
npm install
npm run dev
```
