# 📚 지식 플레이스 (Knowledge Place)

재미있는 게임을 통해 학습하는 혁신적인 교육 플랫폼입니다.

## 🎮 게임 모드

### ⚡ 스피드 퀴즈
- **60초 시간 제한**: 제한된 시간 내 최대한 많은 문제 해결
- **빠른 정답 시 보너스 점수**: 신속한 답변에 대한 추가 점수
- **연속 정답 시 추가 보너스**: 연속 정답 시 콤보 보너스 시스템

### 💪 서바이벌 모드
- **3개의 생명으로 시작**: 초기 생명 3개 제공
- **틀릴 때마다 생명 감소**: 오답 시 생명 차감
- **무한 도전 가능**: 생명이 남아있는 한 계속 플레이

### 📖 연습 모드
- **시간 제한 없음**: 여유롭게 학습 가능
- **상세한 해설 제공**: 각 문제마다 자세한 설명
- **이전 문제 복습 가능**: 앞뒤 문제 자유 이동

### 🎯 도전 모드
- **어려운 문제 10개 선별**: 엄선된 고난도 문제
- **문제당 30초 제한**: 각 문제별 시간 제한
- **시간 보너스 및 등급 시스템**: 남은 시간에 따른 보너스

### 🧀 치즈 헌터
- **뱀 게임 + 퀴즈**: 클래식 뱀 게임과 한국어 퀴즈의 결합
- **아이템 시스템**: 다양한 아이템과 디버프
- **명예의 전당**: 최고 점수 기록

```

    └── world_history.ts  # 세계사
```

### 문제 추가 방법

1. **해당 학급과 교과목 파일 열기**
   - 예: 중학교 수학 문제 추가 → `src/data/questions/middle/math.ts`

2. **문제 객체 추가**
```typescript
{
  id: 'mid_math_11',                    // 고유 ID (학급_교과_번호)
  question: '문제 내용을 입력하세요',      // 문제
  options: ['선택지1', '선택지2', '선택지3'], // 선택지 (3-4개)
  answer: '정답',                       // 정답 (options 중 하나)
  explanation: '해설을 입력하세요',        // 해설
  grade: 'middle',                     // 학급 ('elementary', 'middle', 'high')
  encouragement: '격려 메시지'           // 정답 시 격려 메시지
}
```

3. **문제 작성 팁**
   - **명확한 질문**: 애매하지 않고 명확한 질문 작성
   - **적절한 난이도**: 해당 학급 수준에 맞는 난이도 설정
   - **정확한 정답**: 선택지 중 하나와 정확히 일치하는 정답
   - **상세한 해설**: 왜 그것이 정답인지 명확한 설명
   - **격려 메시지**: 학습자를 격려하는 긍정적인 메시지

4. **ID 명명 규칙**
   - 형식: `{학급}_{교과}_{번호}`
   - 학급: `elem` (초등), `mid` (중등), `high` (고등)
   - 교과: `kor` (국어), `math` (수학), `eng` (영어), `sci` (과학), `hist` (한국사), `world` (세계사)
   - 예: `elem_math_15`, `high_eng_8`

## 🚀 배포 방법

### Netlify 배포
1. 프로젝트를 GitHub에 푸시
2. Netlify에서 새 사이트 생성
3. GitHub 저장소 연결
4. 빌드 설정:

   - Publish directory: `dist`
5. 배포 완료
